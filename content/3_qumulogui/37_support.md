The Support section provides access to Qumulo's comprehensive support ecosystem, software upgrades, and helpful resources. This is your gateway to expert assistance and platform maintenance tools.

## Available Support Options

The Support section includes:
- **Qumulo Care Portal** - Access to knowledge base and support cases
- **Software Upgrades** - Cluster upgrade management
- **Documentation Links** - Direct access to technical resources

## Qumulo Care Portal

Your primary resource for support and product information:

### What Qumulo Care Provides

**Support Services:**
- **Open support cases** with Qumulo technical experts
- **Track case progress** and receive updates
- **Access priority support** based on your service level

**Knowledge Resources:**
- **Comprehensive knowledge base** with solutions and best practices
- **Product documentation** and technical guides
- **Community forum** for peer-to-peer assistance

**Product Information:**
- **Latest software releases** and release notes
- **Feature roadmap** and upcoming capabilities
- **Training materials** and certification programs

Access Qumulo Care at: [care.qumulo.com](http://care.qumulo.com/)

## Software Upgrade Management

Manage cluster software updates directly from the interface:

![Software Upgrade Interface](/static/images/qumulogui/37_01.png)

### Upgrade Process
1. **Download** the latest Qumulo Core release from Nexus
2. **Upload** the upgrade file using the local path selector
3. **Plan** the upgrade window with minimal service disruption
4. **Execute** the upgrade with built-in safety checks
