# Section 3: Qumulo GUI Overview

Welcome to your exploration of the Qumulo Cloud Native (CNQ) user interface. This section introduces you to the powerful web-based management console that serves as your primary tool for administering Qumulo file systems in the cloud.

The Qumulo GUI provides a comprehensive, intuitive interface for managing every aspect of your storage environmentâ€”from basic file system operations to advanced analytics and security configurations. Whether you're monitoring performance, configuring replication, or managing user access, the GUI centralizes these capabilities in a user-friendly dashboard.

## Learning Objectives

By the end of this section, you will be able to:

- **Navigate** the Qumulo management interface and understand its layout and main components
- **Identify** key dashboard elements and interpret system status indicators
- **Explore** file system management capabilities
- **Understand** snapshot and replication functionality overview
- **Recognize** user and access management features
- **Locate** monitoring, analytics, and reporting tools
- **Familiarize yourself** with system configuration and administrative settings
- **Access** documentation and support resources directly from the interface

## What This Section Covers

This section provides a guided tour of the Qumulo GUI, focusing on:

### Core Interface Components
- Dashboard overview and navigation
- Menu structure and organization
- Status indicators and alerts
- User interface customization options

### Key Functional Areas
- **File System Management**: Directory browsing, file operations, and quota management
- **Storage Analytics**: Real-time performance metrics and capacity utilization
- **Data Protection**: Snapshot scheduling, and replication status
- **Access Control**: User management, permissions, and authentication settings
- **System Administration**: Cluster configuration, networking, and maintenance tools

### Practical Exploration
Each subsection includes hands-on activities where you'll:
- Navigate to specific areas of the interface
- Observe live data and metrics from your workshop cluster
- Practice common administrative tasks
- Identify where to find detailed configuration options
